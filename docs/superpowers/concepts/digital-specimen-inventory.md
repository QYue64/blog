# 数字标本室实现清单

## 已确认的视觉基准

- 首页：`digital-specimen-home.png`，原始尺寸 1586×992。
- 内容页：`digital-specimen-content.png`，原始尺寸 1159×1358。
- 页面背景锁定为浅冰蓝 `#EDF4FF`，不替换为白色、米白或深色。

## 首屏文案白名单

- `QYUE://HOME`
- `文章`
- `关于`
- `链接`
- `菜单`
- `好奇心，有自己的 homepage.`
- `AI · HOMELAB · DEVOPS`
- `NOW.TXT`
- `正在研究如何让 AI 工具真正融入日常工作流。`
- `STATUS`
- `BUILDING`
- `DOCUMENTING`
- `SHARING`

不增加眉题、徽章、指标、搜索框、主按钮或解释性副标题。

## 色彩与表面

| 角色 | 色值 | 用法 |
| --- | --- | --- |
| Ice | `#EDF4FF` | 全站固定背景 |
| Ink | `#172030` | 正文、边框、硬阴影 |
| Lime | `#C9FF5C` | 首屏有机色块、章节提示 |
| Violet | `#7656EF` | 链接、进度、当前状态 |
| Pink | `#FFB8DB` | 状态窗口、代码栏、悬停预览 |
| White | `rgba(255,255,255,.87)` | 普通标本窗口 |
| Muted | `#596578` | 次要正文 |
| Line | `#9BA7BA` | 次要分隔线 |

没有图片叠色、渐变、发光或玻璃拟态。首页与文章页都不依赖封面图片。

## 字体

- 展示标题：系统粗体无衬线，中文优先 `PingFang SC`，字距约 `-.075em`，行高约 `.88–.96`。
- `homepage.`：Georgia/宋体衬线斜体，正常字重。
- 导航、编号、元数据、窗口栏和代码：系统等宽字体。
- 正文：系统中文无衬线，文章列约 `68ch`，行高约 `1.8`。
- 所有按钮与菜单显式设置字体尺寸和字重，不依赖浏览器默认值。

## 页面结构

### 首页

安静导航 → 超大标题与有机色块 → 两个标本窗口 → 关键词轨道 → 最新文章开放条带 → 三条最近记录 → 关于与外部链接双区 → 页脚。

### 文章索引

大标题与文章总数图形 → 排序元数据 → 开放日志表格。桌面为编号、标题、分类、日期四列；移动端为编号、标题、日期三列。

### 阅读页

固定阅读进度 → 文章标题区与元数据窗口 → 左侧返回、中间正文、右侧目录 → 浅色代码块 → 上下篇。正文区不使用漂浮或循环动画。

## 组件与容器

- `SpecimenWindow`：1px Ink 边框，6–8px 硬阴影，白色或 Pink 两种变体。
- 日志列表：开放行与分隔线，不使用卡片。
- 代码块：白色主体、Pink 标题栏、Ink 边框。
- 移动菜单：唯一允许的大型覆盖层；必须支持键盘和焦点恢复。
- 圆角只用于移动菜单或必要触控容器，主要视觉保持直角。

## 图标清单

| 图标 | 形式 | 用法 |
| --- | --- | --- |
| 菜单 | 三条等宽横线 | Header 菜单按钮，Ink，24px |
| 关闭 | 两条相交直线 | 移动菜单与窗口装饰，Ink，20–24px |
| 向右箭头 | 简洁直线箭头 | 日志行入口，Violet/Ink，16–20px |
| 向左箭头 | 简洁直线箭头 | 返回文章列表，Violet，16–20px |
| 复制 | 叠放矩形线框 | 代码复制，Ink，16px |

交互图标使用仓库现有 Icon 组件或清晰的内联 SVG，保持 1.5–2px 直线描边；窗口最小化和方框仅为装饰并标记 `aria-hidden`。

## 响应式与动效

- 桌面基准 1440×900；移动基准 390×844。
- 首屏标题移动端不超过三行，窗口移动到标题下方且不遮挡正文。
- 文章目录在窄屏隐藏；代码块允许自身横向滚动，页面不横向溢出。
- 关键词轨道、有机色块和窗口只做轻量循环/视差；`prefers-reduced-motion` 下停止。
- 日志悬停和键盘聚焦必须得到相同反馈。
